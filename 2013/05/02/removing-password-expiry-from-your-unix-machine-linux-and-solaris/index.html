Removing password expiry from your Linux/Unix machine

Here’s a quick one. If you have a server and you do not want the password for a user to expire (as it can screw some things up while its active), you need to execute the following commands as root: passwd -x -1

where is the username whose password expiry you wish to remove. For instance, in my case, if username is dm014, I executed: passwd -x -1 dm014 I have tested this and it works flawlessly on most Linux and Solaris operating systems.
